# ML-helpers

A collection of helpful bits of code and tooling for ML workflows.

## Overview

This repository contains reusable utilities and tools aimed at making common machine learning tasks easier. The goal is to provide practical, well-organized helpers that can be dropped into ML projects to speed up experimentation and productionization.

## Contents

As the repository grows, you can expect to find helpers covering areas such as:

- **Data loading & preprocessing** – utilities for reading, cleaning, and transforming datasets
- **Training utilities** – learning-rate schedulers, early stopping, checkpointing, and logging helpers
- **Evaluation** – metrics, visualisation tools, and result-reporting helpers
- **Model utilities** – weight initialisation, layer builders, and common architectural patterns
- **Experiment management** – configuration handling and reproducibility helpers

## Getting Started

Browse the relevant subdirectory for the kind of helper you need. Each module includes its own documentation and usage examples.
